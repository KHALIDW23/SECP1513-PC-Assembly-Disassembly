## üìã Session Overview

### **Objective:**
Gain practical, hands-on experience with computer hardware components through systematic disassembly and reassembly of a desktop computer system.

### **Activities Performed:**
1. **Component Identification**: Recognized and understood the function of each internal component
2. **Safe Disassembly**: Systematically dismantled the PC while observing ESD safety precautions
3. **Reassembly Process**: Correctly reinstalled all components in proper sequence
4. **System Testing**: Verified successful boot and operational stability
5. **Troubleshooting**: Diagnosed and resolved common hardware issues

### **Components Worked With:**
- **Motherboard**: Base system installation and component integration
- **CPU & Cooling System**: Processor installation and thermal management
- **RAM Modules**: Memory configuration and installation
- **Storage Devices**: HDD/SSD mounting and connection
- **Power Supply**: Electrical connections and cable management
- **Expansion Cards**: GPU and other peripheral card installation
- **Casing & Cooling**: Physical assembly and airflow optimization

## üí° Key Takeaways

### **1. Attention to Detail Matters**
- Small oversights (forgotten power connections, improperly seated RAM) can prevent system boot
- Cable management affects both aesthetics and airflow/thermal performance
- Component orientation and alignment are critical for proper function

### **2. Systematic Approach is Essential**
- Following a logical sequence saves time and prevents errors
- Documentation of disassembly helps ensure correct reassembly
- Testing at each major step simplifies troubleshooting

### **3. Theory vs. Practice Bridge**
- Textbook diagrams became tangible, understandable components
- Abstract concepts like "bus" and "interface" became physical connectors
- Performance specifications took on practical meaning

### **4. Confidence Building**
- Overcoming initial apprehension about handling delicate components
- Successfully troubleshooting and resolving issues
- Transforming theoretical knowledge into practical capability
